HTLM & CSS Basic Summary

HTML Summary

HTML (HyperText Markup Language) us used to define your page content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, content and then an opening tag
  • You can also have void (empty) elements like images
  • You can also configure elements with attributes

    • There's a long list od available elements but you'll gain experience over time, no worries! Learn more about all available HTML elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content.

  • Style are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplication, you typically use global styles (via the "style" element) instead
  • Alternatively, you work external stylesheets files which you "link" to
  • When working with CSS, conecpts like "inheritance" and "specificity" should be understood

Learn more about all about CSS properties and values on the MDN CSS property reference.
